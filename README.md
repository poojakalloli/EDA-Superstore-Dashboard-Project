# 📊 EDA Superstore Dashboard Project

## 🚀 Project Overview

This project focuses on **Exploratory Data Analysis (EDA)** and building **interactive dashboards** using Superstore data.

The main goal is to analyze business performance and extract meaningful insights from sales, customers, and inventory data.

---

## 📌 Dashboards Included

### 🔹 Superstore Performance Dashboard

* Top 10 Products by Sales
* Sales by Category
* Payment Mode Distribution
* Sales Over Time

## 📈 Key Insights

* Technology category generates highest sales
* High discounts reduce profit margins
* Some regions have high sales but low profit
* Inventory management needs optimization
